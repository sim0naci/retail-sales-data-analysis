# Retail Sales Data Analysis

## Project Overview

This project explores a retail sales dataset to understand customer purchasing behavior and identify business insights. The analysis focuses on sales performance, customer demographics, and product categories using Python and data analysis libraries.

The goal of this project is to practice the full data analysis workflow, including data cleaning, transformation, and exploratory data analysis.

---

## Dataset

The dataset contains transactional retail data with the following fields:

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

Each row represents a single purchase transaction.

## Analysis Performed

The analysis includes:

* Data cleaning and preprocessing
* Handling missing values and correcting data types
* Creating new calculated columns
* Filtering and segmenting the dataset
* Aggregating data using groupby
* Identifying sales trends and patterns

---

## Key Questions Explored

Some of the business questions explored in this project include:

* Which product categories generate the highest revenue?
* What are the sales trends over time?
* Which customer segments contribute the most to revenue?
* How does purchasing behavior vary by age or gender?
* What products generate the highest sales volume?
